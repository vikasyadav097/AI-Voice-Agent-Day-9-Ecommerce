## Overview
Build a **D&D-style voice Game Master** that runs an interactive story in a specific universe and guides the player through an adventure.

## Primary Goal (Required)
Build a voice-only game master that:
1. Uses a clear Game Master persona
2. Drives an interactive story using voice
3. Maintains continuity with chat history
4. Runs a short session (8-15 exchanges)
5. Has basic UI showing GM text and player speech

## Advanced Goals (Optional)
1. JSON World State - Track characters, events, locations
2. Player Character Sheet & Inventory
3. Simple Mechanics - Dice rolls and checks
4. Multiple Universes via presets
5. Save & Resume Game
